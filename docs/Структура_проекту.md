# Структура проекту Eurogranite

## Загальний огляд
Проект являє собою статичний сайт компанії Eurogranite - виробника преміальної гранітної продукції для європейських ринків. Сайт створено на основі Create React App з підтримкою Tailwind CSS.

## Особливості поточної реалізації

### Архітектура додатку

#### SPA (Single Page Application) з React Router
- **BrowserRouter** - основний роутер в App.js
- **Routes/Route** - опреділення маршрутів для 3 сторінок
- **Link компоненти** - навігація без перезавантаження сторінки
- **useLocation hook** - відстеження активної сторінки для підсвічування

#### Atomic Design
- **Atoms:** Button, AnimatedCounter, OptimizedImage - базові UI елементи
- **Molecules:** LanguageSwitcher, GraniteCard - комбінації атомів
- **Organisms:** Header, Hero, GraniteColorSection, Footer - складні секції

#### State Management (Zustand)
- **Language Store:** глобальна мова інтерфейсу (UA/EN/DE/PL) з localStorage
- **Modal Store:** управління відкриттям/закриттям модальних вікон граніту
- **Granite System Store:** НОВИЙ комплексний стор для:
  - Галерея текстур (відкриття/закриття, навігація, завантаження)
  - Система порівняння (до 4 текстур одночасно, режими перегляду)
  - Мокапи візуалізації (застосування текстур на зони, drag & drop, історія)
  - UI стан (активна вкладка, налаштування, тема)

#### Маршрутизація та компоненти
- **`/`** → Home.jsx (Header + Hero + GraniteColorSection + ModalManager)
- **`/products`** → Products.jsx (Header + заголовок + GraniteColorSection + ModalManager)  
- **`/contact`** → Contact.jsx (Header + контактна інформація)
- **Footer** - окремо в App.js поза роутами

### Багатомовність
- Підтримка українській (UA) та англійській (EN) мов
- Реалізована через Zustand store (languageStore)
- Динамічне перемикання мови без перезавантаження

### Структура сторінок
1. **Home** - Головна сторінка з Hero секцією
2. **Products** - Каталог гранітної продукції
3. **Contact** - Форма зворотного зв'язку та контакти

### Ключові компоненти
1. **Header** - Навігація з перемикачем мови
2. **Hero** - Головна секція з презентацією компанії та анімованою статистикою
3. **GraniteColors** - ОНОВЛЕНИЙ секція з типами граніту + інтеграція з галереєю текстур
4. **GraniteModal** - Модальне вікно з деталями граніту
5. **ModalManager** - Управління модальними вікнами
6. **LanguageSwitcher** - Перемикач мови
7. **Button** - Базовий атомарний компонент
8. **AnimatedCounter** - Анімований лічільник для статистики (15+, 500+, 25)
9. **OptimizedImage** - Компонент зображень з lazy loading і responsive підтримкою
10. **TextureThumbnail** - НОВИЙ компонент для відображення мініатюр текстур граніту

### Типи гранітної продукції (ОНОВЛЕНО з текстурами)
1. **Чорний граніт** (TOP вибір) - 1 текстура
2. **Сірий граніт** (Універсальний) - 1 текстура  
3. **Червоно-коричневий граніт** (Унікальний) - 7 текстур:
   - Classic Red-Brown, Mahogany, Rustic, Copper, Burgundy, Autumn, Terra Cotta
4. **Рожево-сірий граніт** (Елегантний) - 1 текстура
5. **Зелений граніт** (Природний) - 1 текстура
6. **Лабрадорит** (Містичний) - 1 текстура

**Загалом:** 12 текстур з повними мультимовними описами та технічними характеристиками
